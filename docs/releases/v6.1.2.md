# Release Notes — Transkryptor v6.1.2

Date de préparation : 2026-06-25
Branche cible : `main`

## Résumé

Transkryptor v6.1.2 est un hotfix de configuration pour aligner les nouvelles
installations sur la liste de modèles Cloud Temple actuellement attendue. Le
pipeline transcription / analyse / synthèse / diarisation reste inchangé.

## Changements

- `.env.example` expose désormais uniquement cette allowlist :
  `qwen3.6:27b,mistral-small4:119b,qwen3.6:35b-a3b,gemma4:31b`.
- Les versions projet, fallbacks client et fallback serveur sont alignés sur
  `6.1.2`.
- Les URLs d'installation README pointent vers le tag `v6.1.2`.
- `changelog.md` conserve le format Keep a Changelog 1.1.0 : section
  `Unreleased`, date ISO, types groupés et liens de comparaison.

## Impact utilisateur

Les nouveaux déploiements créés depuis `.env.example` proposent les modèles
dans l'ordre demandé, avec `qwen3.6:27b` comme modèle sélectionné par défaut.
Les fichiers `.env` existants ne sont pas modifiés automatiquement.

## Commande d'installation cible

```bash
curl -fsSL https://raw.githubusercontent.com/Lesur-ai/transkryptor/v6.1.2/scripts/install.sh | bash
```

## Validation recommandée

- `node --check src/server/server.js`
- `node --check src/client/js/apiService.js`
- `node --check src/client/js/state.js`
- JSON valide pour `package.json` et `package-lock.json`
- `bash -n scripts/install.sh`
- Smoke test HTTP `GET /api/version` avec serveur local
