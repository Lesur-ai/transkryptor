Date de préparation : 2026-06-25

Transkryptor v6.1.1 est un hotfix documentation + installation pour la release
visuelle v6.1.0. Le pipeline transcription / analyse / synthèse / diarisation
reste inchangé.

- Version projet, fallbacks client et fallback serveur alignés sur `6.1.1`.

## Script d'installation

Commande cible :

```bash
curl -fsSL https://raw.githubusercontent.com/Lesur-ai/transkryptor/v6.1.1/scripts/install.sh | bash
```

Le script :

- installe sur macOS dans `~/Applications/Transkryptor` ;
- installe sur Linux dans `${XDG_DATA_HOME:-~/.local/share}/transkryptor` ;
- copie `.env.example` vers `.env` si le fichier n'existe pas ;
- lance `npm ci` ;
- démarre le service sur `http://localhost:3000` ;
- attend `/api/version` puis ouvre le navigateur par défaut.

Variables utiles : `TRANSKRYPTOR_INSTALL_DIR`, `TRANSKRYPTOR_REF`,
`TRANSKRYPTOR_PORT`, `TRANSKRYPTOR_OPEN_BROWSER=0`.

## Windows

La recommandation immédiate est d'utiliser WSL avec le même script bash.
Pour un support natif Windows, la suite propre serait un script PowerShell
installant dans `%LOCALAPPDATA%\Transkryptor`, lançant `npm ci`, démarrant le
service Node et ouvrant `http://localhost:3000`.
